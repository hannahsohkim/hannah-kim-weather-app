# Weather App

### Live Demo: 
https://hannah-weather-app.herokuapp.com/

### Description
A React application that interacts with the MetaWeather API. 

### How to interact with the app
Type in a city to see the weather data displayed for the current day. 

### Technologies used 
- Javascript
- React.js 
- Axios
- Styled Components
- Heroku 
- MetaWeather API
